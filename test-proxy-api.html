💻 代码示例

原来的方式（代理访问）：

// 直接替换URL const proxyUrl = url.replace('https://labubuwallpaper.com', '/download-proxy'); const response = await fetch(proxyUrl);

新的方式（接口访问）：

// 方式1：使用proxyApi服务 import proxyAPI from '@/services/proxyApi'; const proxyUrl = proxyAPI.getProxyUrl(originalUrl); const response = await fetch(proxyUrl); // 方式2：直接下载 const success = await proxyAPI.downloadProxyResource(originalUrl, filename); // 方式3：获取Blob const blob = await proxyAPI.getProxyBlob(originalUrl);
